# EventEase

A marketplace connecting event planners with local service providers.

## Problem

Finding and coordinating multiple service providers for events is time-consuming and fragmented.

## Solution

A wizard-driven platform that matches event planners with curated service providers based on location, budget, and service needs.

## Core Users

- **Event Planners** - People organizing events who need services
- **Service Providers** - Businesses offering event-related services

## Key Features

- Service provider registration and profiles
- Event wizard for capturing requirements
- Smart matching based on location, budget, and availability
- Service categorization (catering, photography, venues, etc.)
