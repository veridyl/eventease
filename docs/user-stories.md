# User Stories

## Event Planner Stories

### Account & Profile
- **US-EP-1**: As an event planner, I want to create an account so I can save my searches and preferences.

### Event Wizard
- **US-EP-2**: As an event planner, I want to specify my event type so I see relevant service categories.
- **US-EP-3**: As an event planner, I want to enter my location so I find nearby providers.
- **US-EP-4**: As an event planner, I want to set my budget so I see affordable options.
- **US-EP-5**: As an event planner, I want to select multiple service types so I can find all providers I need.

### Discovery
- **US-EP-6**: As an event planner, I want to see providers grouped by service type so I can compare within categories.
- **US-EP-7**: As an event planner, I want to view provider profiles so I can assess their work.
- **US-EP-8**: As an event planner, I want to filter by rating and price so I find the best match.

### Engagement
- **US-EP-9**: As an event planner, I want to contact providers directly so I can discuss my needs.
- **US-EP-10**: As an event planner, I want to save favorite providers so I can compare later.

---

## Service Provider Stories

### Account & Profile
- **US-SP-1**: As a provider, I want to create a business account so I can list my services.
- **US-SP-2**: As a provider, I want to add my business details so planners know about me.
- **US-SP-3**: As a provider, I want to upload portfolio images so planners see my work.

### Services
- **US-SP-4**: As a provider, I want to list my services with pricing so planners know my rates.
- **US-SP-5**: As a provider, I want to select service categories so I appear in relevant searches.
- **US-SP-6**: As a provider, I want to define my service area so I get local inquiries.

### Availability
- **US-SP-7**: As a provider, I want to manage my calendar so planners see my availability.

### Inquiries
- **US-SP-8**: As a provider, I want to receive inquiry notifications so I can respond quickly.
- **US-SP-9**: As a provider, I want to view planner event details so I can prepare quotes.

---

- US-SP-1, US-SP-2, US-SP-4, US-SP-5, US-SP-6

**Phase 2**:
