# Requirements

## Functional Requirements

### Event Planner Features

| ID | Requirement |
|----|-------------|
| EP-1 | Register/login to platform |
| EP-2 | Complete event wizard (type, date, location, budget, services needed) |
| EP-3 | View matched service providers by category |
| EP-4 | Filter results by price, rating, availability |
| EP-5 | Contact service providers |
| EP-6 | Save/bookmark providers |
| EP-7 | View provider profiles and portfolios |

### Service Provider Features

| ID | Requirement |
|----|-------------|
| SP-1 | Register/login to platform |
| SP-2 | Create business profile (name, description, contact, location) |
| SP-3 | List services with categories and pricing |
| SP-4 | Upload portfolio (images, videos) |
| SP-5 | Set service areas (locations covered) |
| SP-6 | Manage availability calendar |
| SP-7 | Receive and respond to inquiries |

### Matching System

| ID | Requirement |
|----|-------------|
| MS-1 | Match providers by service category |
| MS-2 | Filter by location/service area |
| MS-3 | Filter by budget range |
| MS-4 | Rank by relevance (rating, reviews, response time) |

## Non-Functional Requirements

| Category | Requirement |
|----------|-------------|
| Performance | Page load < 3 seconds |
| Availability | 99% uptime |
| Security | Encrypted data, secure authentication |
| Scalability | Support 10,000+ providers |
| Mobile | Responsive design |
